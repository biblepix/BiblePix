# ENGLISH #

The BiblePix Installer (BiblePix-Installer.tcl) should only be executed once. 
After that, running BiblePix Setup is recommended: just double-click on the file ~/Biblepix/biblepix-setup.tcl

Recommended place for Git installation:
  Linux: /home/[MYNAME]/Biblepix/
  Windows: C:\Users\[MYNAME]\AppData\Local\Biblepix\


# DEUTSCH #

Der Installer (BiblePix-Installer.tcl) sollte nur das erste Mal ausgeführt werden, danach
mit Doppelklick auf die Datei ~/Biblepix/biblepix-setup.tcl das Einrichtungsprogramm
betätigen.

Empfohlener Installationsort für Git-Installation:
  Linux: /home/[MYNAME]/Biblepix/
  Windows: C:\Users\[MEINNAME]\AppData\Local\Biblepix\
